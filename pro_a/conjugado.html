• Multiplicación por el Conjugado en Límites

La técnica de multiplicación por el conjugado es muy útil cuando estamos trabajando con límites que involucran raíces cuadradas. Este método permite simplificar la expresión y eliminar la indeterminación. La idea consiste en multiplicar y dividir por el conjugado de una expresión para simplificar las raíces y poder calcular el límite.

Pasos para usar la multiplicación por el conjugado:

  • Identificar el límite: Se te da una función que involucra una raíz cuadrada, y un valor al cual (x) tiende, es decir, un límite limx→c f(x).
  • Sustituir el valor de (x): Intentamos sustituir directamente el valor (x = c) en la expresión de la función.
    • Si obtenemos una forma indeterminada como 0/0 o √a - √b = 0, entonces necesitamos multiplicar por el conjugado.
  • Multiplicar y dividir por el conjugado: Multiplicamos y dividimos la expresión por el conjugado de la parte que tiene la raíz cuadrada.
  • Simplificar la expresión: Realizamos las simplificaciones necesarias y, si es posible, cancelamos factores comunes.
  • Reemplazar el valor de (x) en la nueva expresión simplificada: Ahora sustituimos el valor de (x = c) nuevamente para obtener el límite.

Ejemplo 1: Límite con raíz cuadrada

Vamos a calcular el siguiente límite:

limx→4 (√x - 2) / (x - 4)

Solución:

  • Identificar el límite: La función es f(x) = (√x - 2) / (x - 4), y queremos calcular el límite cuando (x) tiende a 4.
  • Sustituir directamente el valor (x = 4):
f(4) = (√4 - 2) / (4 - 4) = (2 - 2) / 0 = 0 / 0

Esto da una forma indeterminada 0/0, por lo que necesitamos multiplicar por el conjugado.

  • Multiplicar y dividir por el conjugado: El conjugado de (√x - 2) es (√x + 2). Entonces, multiplicamos y dividimos la función por (√x + 2):
f(x) = ((√x - 2)(√x + 2)) / ((x - 4)(√x + 2))
  • Simplificar la expresión: Usamos la identidad de diferencia de cuadrados (a - b)(a + b) = a² - b² para simplificar el numerador:
f(x) = (x - 4) / ((x - 4)(√x + 2))
  • Cancelar factores comunes: Podemos cancelar el factor (x - 4) en el numerador y el denominador (siempre que x ≠ 4):
f(x) = 1 / (√x + 2)

Reemplazar el valor (x = 4) en la expresión simplificada:

f(4) = 1 / (√4 + 2) = 1 / (2 + 2) = 1 / 4

Resultado: El límite es 1/4.

limx→4 (√x - 2) / (x - 4) = 1/4

Ejemplo 2: Límite con raíz cuadrada en el numerador y denominador

Vamos a calcular el siguiente límite:

limx→9 (√x - 3) / (√x - 3)

Solución:

  • Identificar el límite: La función es f(x) = (√x - 3) / (√x - 3), y queremos calcular el límite cuando (x) tiende a 9.
  • Sustituir directamente el valor (x = 9):
f(9) = (√9 - 3) / (√9 - 3) = (3 - 3) / (3 - 3) = 0 / 0

Esto da una forma indeterminada 0/0, por lo que necesitamos multiplicar por el conjugado.

  • Multiplicar y dividir por el conjugado: El conjugado de (√x - 3) es (√x + 3). Entonces, multiplicamos y dividimos la función por (√x + 3):
f(x) = ((√x - 3)(√x + 3)) / ((√x - 3)(√x + 3))
  • Simplificar la expresión: Usamos la identidad de diferencia de cuadrados (a - b)(a + b) = a² - b² para simplificar el numerador:
f(x) = (x - 9) / ((√x - 3)(√x + 3))
  • Cancelar factores comunes: Podemos cancelar el factor (√x - 3) en el numerador y el denominador (siempre que x ≠ 9):
f(x) = 1 / (√x + 3)

Reemplazar el valor (x = 9) en la expresión simplificada:

f(9) = 1 / (√9 + 3) = 1 / (3 + 3) = 1 / 6

Resultado: El límite es 1/6.

limx→9 (√x - 3) / (√x - 3) = 1/6
