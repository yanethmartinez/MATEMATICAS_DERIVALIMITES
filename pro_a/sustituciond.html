• Sustitución Directa para Calcular un Límite

La sustitución directa es un método utilizado para calcular límites en cálculo, especialmente útil cuando la función es continua en el punto al cual estamos acercándonos. Consiste en simplemente sustituir el valor al cual la variable se acerca (el valor del límite) en la expresión de la función y verificar si obtenemos un valor determinado.

Pasos para usar la sustitución directa:

  • Identificar el límite: Se te da una función f(x) y un valor al que x tiende, es decir, un límite limx→c f(x).
  • Sustituir el valor c en la función f(x): Reemplaza x por el valor al que tiende x en la función.
  • Comprobar el resultado:
    • Si al sustituir el valor de x en f(x) obtenemos un valor real definido, ese es el valor del límite.
    • Si la sustitución resulta en una forma indeterminada (como 0/0, ∞−∞, ∞/∞, etc.), entonces no se puede calcular el límite por sustitución directa y se debe usar otro método, como factorización, racionalización o la regla de L'Hopital.

Ejemplo 1: Calcular un límite usando sustitución directa

Supongamos que tenemos la siguiente función:

f(x) = 2x + 3

Queremos calcular:

limx→4 (2x + 3)

Solución:

  • Identificar el límite: La función es f(x) = 2x + 3 y estamos buscando el límite cuando x tiende a 4.
  • Sustituir directamente el valor de x = 4:
f(4) = 2(4) + 3 = 8 + 3 = 11

Resultado: El límite es 11.

limx→4 (2x + 3) = 11

Ejemplo 2: Límite con una función racional

Considera la siguiente función:

f(x) = (x² - 4) / (x - 2)

Queremos calcular:

limx→2 (x² - 4) / (x - 2)

Solución:

  • Identificar el límite: La función es f(x) = (x² - 4) / (x - 2) y estamos buscando el límite cuando x tiende a 2.
  • Intentar la sustitución directa: Si sustituimos x = 2 en la función:
f(2) = (2² - 4) / (2 - 2) = (4 - 4) / 0 = 0 / 0

Este es una forma indeterminada 0/0, por lo que no podemos calcular el límite por sustitución directa en este caso. Necesitamos simplificar la expresión antes de calcular el límite.

  • Factorizamos el numerador:
x² - 4 = (x - 2)(x + 2)

Ahora la función se convierte en:

f(x) = (x - 2)(x + 2) / (x - 2)

Podemos cancelar (x - 2) en el numerador y el denominador (siempre que x ≠ 2):

f(x) = x + 2

Ahora podemos calcular el límite sustituyendo directamente x = 2:

limx→2 (x + 2) = 2 + 2 = 4

Resultado: El límite es 4.

limx→2 (x² - 4) / (x - 2) = 4
