• Factorización para Calcular un Límite

La factorización es una técnica importante para calcular límites, especialmente cuando nos encontramos con formas indeterminadas como \( \frac{0}{0} \), que suelen ocurrir en funciones racionales. Esta técnica consiste en factorizar el numerador o el denominador (o ambos) de la expresión de la función, y luego simplificar la expresión para eliminar la indeterminación, permitiendo calcular el límite de manera más fácil.

Pasos para usar la factorización en el cálculo de un límite:

  • Identificar el límite: Se te da una función racional y un valor al cual ( x ) tiende, es decir, un límite limx→c f(x).
  • Sustituir el valor de ( x ): Intentamos sustituir directamente el valor ( x = c ) en la expresión de la función.
    • Si obtenemos una forma indeterminada como 0/0, entonces necesitamos factorizar.
  • Factorizar la expresión: Factorizamos el numerador y/o el denominador para intentar simplificar la expresión.
  • Cancelar factores comunes: Si la factorización da como resultado factores comunes en el numerador y el denominador, los podemos cancelar (siempre que el valor ( x = c ) no haga que esos factores sean 0).
  • Reemplazar el valor de ( x ) en la nueva expresión simplificada: Ahora, con la expresión simplificada, sustituimos el valor de ( x = c ) nuevamente para obtener el límite.

Ejemplo 1: Límite con una función racional

Vamos a calcular el siguiente límite:

limx→2 (x² - 4) / (x - 2)

Solución:

  • Identificar el límite: La función es f(x) = (x² - 4) / (x - 2), y queremos calcular el límite cuando ( x ) tiende a 2.
  • Sustituir directamente el valor ( x = 2 ):
f(2) = (2² - 4) / (2 - 2) = (4 - 4) / 0 = 0 / 0

Esto da una forma indeterminada \( \frac{0}{0} \), por lo que necesitamos factorizar.

  • Factorizar el numerador: El numerador ( x² - 4 ) es una diferencia de cuadrados, y se puede factorizar como:
x² - 4 = (x - 2)(x + 2)

Entonces, la función se convierte en:

f(x) = (x - 2)(x + 2) / (x - 2)
  • Cancelar los factores comunes: Podemos cancelar el factor ( (x - 2) ) en el numerador y el denominador (siempre que ( x no es igual a 2 )): 
f(x) = x + 2

Reemplazar el valor ( x = 2 ) en la expresión simplificada:

f(2) = 2 + 2 = 4

Resultado: El límite es 4.

limx→2 (x² - 4) / (x - 2) = 4

Ejemplo 2: Límite con una función racional más compleja

Vamos a calcular el siguiente límite:

limx→3 (x² - 9) / (x - 3)

Solución:

  • Identificar el límite: La función es f(x) = (x² - 9) / (x - 3), y queremos calcular el límite cuando ( x ) tiende a 3.
  • Sustituir directamente el valor ( x = 3 ):
f(3) = (3² - 9) / (3 - 3) = (9 - 9) / 0 = 0 / 0

Esto también da una forma indeterminada \( \frac{0}{0} \), por lo que necesitamos factorizar.

  • Factorizar el numerador: El numerador ( x² - 9 ) es otra diferencia de cuadrados, y se puede factorizar como:
x² - 9 = (x - 3)(x + 3)

Entonces, la función se convierte en:

f(x) = (x - 3)(x + 3) / (x - 3)
  • Cancelar los factores comunes: Podemos cancelar el factor ( (x - 3) ) en el numerador y el denominador (siempre que ( x no es igual a 3 )): 
f(x) = x + 3

Reemplazar el valor ( x = 3 ) en la expresión simplificada:

f(3) = 3 + 3 = 6

Resultado: El límite es 6.

limx→3 (x² - 9) / (x - 3) = 6
